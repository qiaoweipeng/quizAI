# 消防-理论考试系统

## 系统介绍

一个考试答题系统

## 系统功能

1. 考试功能：

- 可选固定考卷、随机考卷与专项练习 3类。
  - 固定考卷 和 随机考卷 类型一样，只是题目不一样。
  - 类型都是100道单选、40道多选、60道判断题组成。每题0.5分，60分合格，考试时间90分钟。
  - 固定考卷每张考卷的题目都是固定不变的，随机考卷每张考卷的题目都是随机的。

- 专项练习：用户可以指定专门练习某一类题目。如：选择题、判断题。可以指定题目的数量。
  - 专项练习没有分数，最后会给出一个正确率，比如你选了10道题，答对了9道，正确率就是90%。

- 试题每天都会给句鉴定站实时更新。

2. 系统UI：

- UI采用简约风格
- UI目前只适配PC端
- 答题页面有两套UI
  - 默认每次只展示一道题（有上一道或者下一道的切换按钮，也可以选择在答题后自动切换）；
  - 展示所有题目，通过滚动条查看。


3. 其他功能：

- 三种考试模式都是可以选择，要不要每答完一道题就立马判断出对错并给出解析。
- 考试完毕都会给出错题极解析。
- 考题分两种模式，考试模式和练习模式
  - 考试模式 ：固定题目、随机题目
  - 练习模式：可选 题目类型（单选、多选、判断，只能选一种）、考题数量（最少10～所有题目的数量）、考试大纲（可多选）。
- 考试模式支持一键预答题（单选选择a，多选全选，判断选对）

## 需要添加的功能

- 添加用户管理功能，后面可以给用户开通会员，用户就可以使用此系统答题。用户需要使用手机号，验证码登录。
- 添加用户支付功能。（支付宝 或 微信支付）
- 有了 用户管理 与 支付功能 就需要加强系统的安全和健壮性。考试数据与用户数据需要保存在mysql数据库中。
- 专项练习可以选择 章节.(如：水系统、防火巡查、监控室、基础知识 ...)
- AI智能分析用户答题结果，给出建议。
- AI根据用户答题的结果，自动出题。
